KEY PROJECTS

1. Productivity Analysis in workspace

A web application bundled with a desktop client that leverages
the power of web technologies and python to calculate probable
productivity by analysing the time spent on different
applications and web history. It can also be deployed to a
workspace to measure employee productivity for effective
analysis of their competitiveness and suggest courses, if required.
