Introduction to Machine learning

Author: Amit Jha | Sep 14th, 2019

A lot of course teach machine learning online but none of them teach machine learning the way a beginner should be taught. Most of the courses extensively talk about the mathematical constructs from day one without giving time to the new learning to settle in and get a hang of it.

In this blog post I'll teach machine learning the way a beginner should be taught. So let's get started...

There are only 4 steps in any machine learning implementation

  • Hypothesis
  • Prediction
  • Cost function
  • Model Training
We will discuss each of them below

Hypothesis

A hypothesis is a proposed explanation for a phenomenon

A quick Google search gave me this definition of Hypothesis. A proposed explanation?Hmmm, what does that has to do with machine learning? Well, in machine learning we have data(we only talk about csv files here, csv files are similar to excel sheets) which should follow some mathematical equation(assumption, we propose that it does). This mathematical function is what we call a Hypothesis of a model.

Not clear with hypothesis yet? Let's look at an example:
consider this data



Here, as you can see, we have 2 columns representing years of experience and the respective salary of the employee. In ML one row of such data is called a datapoint. Now if we were to plot a scatter plot of this data we would get something like this



Do you see a pattern? I bet you do. This graph loosly resembles a straight line, doesn't it? Now here we propose that this data resembles a straight line. This straight line is our hypothesis.
That is, we assume that a model based on this straight line can predict values from future datapoint that might appear in this dataset.
So, let's draw a line


This line is a hypothesis for a linear regression model

Prediction

Now we use the above discussed hypothesis to do prediction

But how?

Let's talk a bit about maths here
A straight line is represented as y = mx + c. Here, m is the slop of the line, c is the y-intercept, x is the dependent variable and y is what we get when we plug in the value of m, c and x. Now, if I were to ask you to draw a straight line on this scatter plot your would probably try to find the best value of m and c so draw this line. That's exactly what a computer does. To begin with, the computer chooses a random value of m and c and then feed in x ( which in this case is years of experience). These three values gives us y which is out prediction from this proposed model based on a straight line.
